# SiPlan_desain-thinking-project
SiPlan adalah aplikasi perencanaan belanja yang dirancang khusus untuk pedagang mikro dan keliling. Aplikasi ini membantu pengguna merencanakan pembelian, menghitung profit otomatis, dan mengelola proses perencanaan belanja dengan efisien melalui fitur catatan belanja dan dapat konfirmasi stok via WhatsApp yang telah terintegrasi.
